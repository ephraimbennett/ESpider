What is ESpider?

ESpider is a web crawling software which indexes any website and returns important SEO analytic information. A full scale java application which requires JRE 20 to run. Finds broken links, response times, asset sizes, HTTP Headers, Site maps, and more.

Easy to install and set up

Get started with the JAR file or pull from my GitHub Repository, No lengthy install pages required - it's good to go!

Generate Sitemaps, Test Broken Links and more.

Interactive visual representations of site structure and links. Customizable and detailed to user specified needs.

Export analytics to multiple formats

Various crawl data can be easily sorted, filtered, and exported to csv, excel, and more.
